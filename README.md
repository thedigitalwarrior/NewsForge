# NewsForge

Rete di siti di news statici (Astro) con contenuti generati da pipeline Claude Agent SDK,
deploy via Ansible su Proxmox/Caddy.

Punto d'ingresso per capire il progetto: `CLAUDE.md` (architettura e convenzioni),
`docs/decisioni.md` (perché le cose sono fatte così), roadmap in fondo a `CLAUDE.md`.

Avvio rapido sviluppo: vedere la sezione "Comandi ricorrenti" in `CLAUDE.md`.
